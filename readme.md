## 项目简介
  - 本项目是基于 Requests 库与 PyQuery 库实现的简单爬虫：
    - 使用 Requests 库获取网页 HTML 内容，PyQuery 库选择器处理 HTML 文本；
    - 通过自定义请求头，模拟浏览器访问；
    - 实现缓存机制，节约爬取成本，提高工作效率和程序容错率。
